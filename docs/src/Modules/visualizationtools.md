# DirUtils.jl

The `DirUtils` module provides utility functions to manage directories, save simulation parameters and results, and load data for Simulated Annealing ABC (sABC) workflows. These functions simplify the management of simulation files and directories.

## Installation

To include the module in your Julia project, add the following lines to your main script:

```Julia
include("./DirUtils.jl")
using .DirUtils
```
